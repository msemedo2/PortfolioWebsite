Crwn Clothing

Crwn Clothing is an E-commerce application that allows users to select clothing items from a mix of item categories, add them to the cart and finally test a card payment method.

The app was created using React, Redux, and the Stripe API for the handling of payments.
It also utilizes Firebase for handling the storage, authentication, and authorization through google or with email and password.

Meet App

A serverless, progressive web app (PWA) built with React following a test-driven development approach (TDD).

The app uses google calendar API to fetch upcoming events for a selected city, and OAuth2 authentication flow.
For the authorization server, it uses AWS Lambda. A service worker allows the app to work offline.
The recharts library is used to visualize data with a scatter graph and a pie chart.
